# PacManLevelEditor

School project 2023<br>
TS + HTML + CSS

## To run:

generate node_modules:<br>
`npm i`

ts -> js :<br>
`npm install -g typescript`<br>
`tsc ts/main.ts -w --outFile static/scripts/main.js --module amd`

run html file:<br>
`index.html`

or run server:<br>
`nodemon server.js`
and open `localhost:3000`
